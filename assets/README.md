# Assets Directory

## Required Files

### Resume
- **File**: `Kofi_Archer_Resume.pdf`
- **Purpose**: Downloadable resume for visitors
- **Action**: Add your professional resume as a PDF file here

### Open Graph Image
- **File**: `og-image.jpg`
- **Purpose**: Social media preview image when sharing your portfolio
- **Recommended Size**: 1200x630 pixels
- **Format**: JPG or PNG
- **Action**: Optional. Current pages use `profile-photo.jpg` so social previews do not point at a missing file.

### Profile Photo
- **File**: `profile-photo.jpg`
- **Purpose**: Your professional headshot displayed on the homepage
- **Recommended Size**: 400x500 pixels (portrait)
- **Format**: JPG
- **Action**: Add a high-quality professional photo

## Tips

- Keep file sizes optimized (compress images without losing quality)
- Use consistent branding across all images
- Ensure profile photo has good lighting and professional appearance
- OG image should include text that's readable at small sizes
