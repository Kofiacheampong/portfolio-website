About Me

I'm a Data Engineer and Cloud Developer who transforms complex data challenges into scalable solutions. With expertise spanning cloud infrastructure, data pipelines, and full-stack development, I help businesses leverage their data and modernize their technology stack.

My Journey

My passion for technology began with curiosity about how things work behind the scenes. This led me through software engineering, data analytics, and infrastructure management, where I discovered my love for building scalable, maintainable systems.

I specialize in building data pipelines, creating full-stack web applications, and deploying solutions across multiple cloud platforms including Oracle Cloud, AWS, and Azure. With experience managing production deployments on Ubuntu servers with Nginx, I ensure applications are secure, performant, and reliable.

Key Achievements:

  • Built and deployed multiple production web applications serving real-time sports analytics
  • Developed AI-powered study tools using transformer models and NLP techniques
  • Designed business intelligence solutions for construction project estimation and pipeline management
  • Implemented custom server configurations with Nginx reverse proxy and SSL/TLS security
